jd-gui
====

## 作用

Java 反编译器 JD-GUI

JD-GUI 是一个用 C++ 开发的 Java 反编译工具，由 Pavel Kouznetsov开发，支持Windows、Linux和苹果Mac Os三个平台。
而且提供了Eclipse平台下的插件JD-Eclipse。JD-GUI不需要安装，直接点击运行，可以反编译jar,class文件。

### 启动命令

> java -jar jd-gui-1.4.0.jar
